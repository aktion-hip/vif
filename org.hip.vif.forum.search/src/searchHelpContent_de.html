Suche nach Beiträgen

Suchbegriffe können kombiniert und mit einer Wildcard (*) ergänzt werden:

Suchbegriff Erklärung
java Findet alle Einträge, welche den Begriff java enthalten.
java junit
java OR junit		 Findet alle Einträge, welche den Begriff java oder junit oder beide enthalten.
java AND junit Findet alle Einträge, welche den Begriff java und junit enthalten.
java* Findet alle Einträge, welche mit dem Begriff java beginnen.

 

Zusätztlich gibt es für Beiträge die Möglichkeit, nach den Feldern author (Autor/in) und id (Frage-Nr.) zu suchen:

Suchbegriff   Erklärung
author:Knuth   Sucht alle Einträge des Autors/der Autorin mit dem Namen Knuth.
id:2\:1.5*   Sucht alle Beiträge einer Gruppe ab einem bestimmten Knoten, in diesem Fall in der Gruppe 2 alle Folgefragen der Frage 1.5. Achtung: Der Doppelpunkt in der Frage-Nr. muss mit "\" geschützt (escaped) werden.

 

Umlaute: Umlaute werden ignoriert, d.h. die Suchbegriffe Müller wie auch Muller finden gleichermassen die Einträge, welche Müller und/oder Muller enthalten.
