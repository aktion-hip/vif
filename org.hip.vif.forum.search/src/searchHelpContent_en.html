Search for contributions

You can combine various query terms and add a wildcard (*):

Query expression Explanation
java Matches contributions that contain the term java.
java junit
java OR junit		 Matches contributions that contain the term java or junit or both.
java AND junit Matches contributions that contain both java and junit.
java* Matches contributions that contain terms that begin with java.

 

In addition you can search for the fields author and id (Question-No.):

Query expression   Explanation
author:Knuth   Matches contributions that contain the name Knuth in the author field.
id:2\:1.5*   Matches all contributions of a discussion group beyond a specified node, in this case in group 2 all follow up questions of question 1.5. Note: The colon in the question No. has to be escaped by "\".
